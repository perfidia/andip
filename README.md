AnDiP
=====

Description
-----------

Dictionary for natural language generation. It contains data about declension, 
conjugation, ... (mainly for Polish an English language).

Installation
------------

### Simple

    python setup.py install

### Using eggs

    python setup.py bdist_egg
    cd dist
    easy_install <package_name>
